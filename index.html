About the Course

The origins of this project trace back to a nifty discussion on Strong AI where Sergio Pissanetzky shared his vision for Strong AI and offered to teach a course on causal theory and write instruction manuals on how to build a causal machine (from a physics background/persepective). This website serves as a permanent and more accessible resource for anyone interested in building the causal machine prototype as conceived by Sergio.

Stay tuned for more updates on the Course Syllabus and schedules.
